# Accounting App

Technical Test PT Trans Berjaya Khatulistiwa

## Tech Stack

- Laravel 12
- MySQL
- Bootstrap 5
- Laravel Excel
- ApexCharts

## Features

- Category CRUD
- COA CRUD
- Transaction CRUD
- Profit Loss Report
- Export Excel
- Dashboard
- Search
- Pagination

## Installation
- composer install
- change .env.example to .env
- php artisan key:generate
- php artisan migrate --seed
- npm install
- npm run build
- php artisan serve
